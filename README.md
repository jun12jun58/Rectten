# Rectten 2026.7.20 ~ 2026.7.21
정보 프로그램 1-1

이름: Rectten
플레이 방법:
1. 마우스를 드래그하거나 화면을 누른채로 움직여서 직사각형 또는 정사각형을 만든다.
2. 만약 그 사각형 안에 있는 숫자들의 합이 10이라면 드래그를 그만두었을 때 그 사각형 영역의 블록들이 부서진다.
3. 부서진 블록들 위에 있는 블록들이 내려와서 빈자리를 매꾸고 맨 위에서 새로운 블럭이 다시 나타난다.

-폭탄-
드래그해서 만든 사각형 안에서는 0으로 계산되고 드래그를 그만두었을 때 폭탄 자리를 중심으로 하여 3*3 영역에 있는 블럭을 부순다.
또는 그냥 클릭 또는 터치해서 작동시킬 수 있다.
